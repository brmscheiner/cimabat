CSS file

All CSS files stored in folder 'CSS', color style stored in folder 'skin'. However, you'd just need to edit the style.css file. It contains the styling elements for all the pages. The stylesheet is well structured and one can find any style element by looking at the very top of the stylesheet which shows all the contents of the stylesheet.

  1. bootstrap.css ( Bootstrap v3.0.0 style )
  2. overwrite.css ( Overwrite Bootstrap v3.0.0 style )
  3. cslider.css ( Style for index.html slider )
  4. nojs.css ( Clear js cslider )
  5. flexslider.css ( Flexslider style for index1.html and detail.html )
  6. font-awesome.css ( icon style )
  7. font-awesome-ie7.css ( icon style for ie )
  8. font-awesome-set.css ( custom icon style )
  9. prettyPhoto.css ( Zoom image )
  10. style.css ( Contains all the styling elements that are used template layout )

Contact setting

Step 1

There are 2 PHP files stored in 'contact' folder. Both files used for the contact form. To make the contact form work properly you'll be required to edit config.php. "change nce18cex@gmail with your email address"

File directory : contact/config.php 

<?php
// To
define("WEBMASTER_EMAIL", 'nce18cex@gmail.com');
?>
Step 2

Edit contact.php for change email subject. look for $subject = "BekkoQ"; and change BekkoQ with your subject

File directory : contact/contact.php 

<?php
/*
Credits: Bit Repository
URL: http://www.bitrepository.com/
*/

include 'config.php';

error_reporting (E_ALL ^ E_NOTICE);

$post = (!empty($_POST)) ? true : false;

if($post)
{

$name = stripslashes($_POST['name']);
$email = trim($_POST['email']);
$subject = "Layapan";
$message = stripslashes($_POST['message']);


$error = '';



if(!$error)
{
$mail = mail(WEBMASTER_EMAIL, $subject, $message,
     "From: ".$name." <".$email.">\r\n"
    ."Reply-To: ".$email."\r\n"
    ."X-Mailer: PHP/" . phpversion());


if($mail)
{
echo 'OK';
}

}


}
?>
